# 🚀 るーにゃ自動コンテンツ生成 - トレンド対応改善版

## 📊 改善概要

2025-2026年のInstagramトレンド分析を踏まえて、`main.py` のシナリオ生成ロジックを大幅改善しました。

**改善前の課題:**
- 「いいね」重視で、実際のアルゴリズムが求める「保存」「シェア」を考慮していない
- 汎用的すぎて、保存ポイントが明確でない
- インタラクティブ要素がない（視聴者が「参加」できない）
- トレンドキーワード・ハッシュタグの配置が最適でない

### 2️⃣ **2025-2026年Instagramアルゴリズムに対応**

#### ✅ 「保存」重視の設計
**アルゴリズム事実:** 「いいね」よりも「保存数」「シェア数」がリーチを決定

実装内容：
- 毎回、必ず「このコンテンツが後で見返したくなる理由」を明示
- 参考になる情報・情報価値を優先
- 視聴者が「ブックマークしておきたい」と思うコンテンツ設計

**具体例:**
```
保存ポイント: 
✅ 新大久保の穴場カフェ3選（住所・営業時間付き）
✅ このメイクのコツ＆使用色（買える場所のリンク）
✅ 一人暮らしの狭い部屋を広く見せるテク5つ
```

#### ✅ 「シェア」重視の設計
**アルゴリズム事実:** DMでシェアされると「強力な拡散性」と認定

実装内容：
- 友達に送りたくなるような驚き・共感・笑いを組み込む
- 「あ、わかる！」となる瞬間を意図的に作る
- 予期しない展開やオチを用意

**具体例:**
```
シェア可能性:
✅ 「私だけじゃなかった」という共感（あるあるネタ）
✅ 予期しない結末やオチ
✅ 思わず笑ってしまう友達との会話シーン
```

#### ✅ インタラクティブ要素（2026年トレンド）
**アルゴリズム事実:** 「指を置く」だけでなく「指を動かす」コンテンツが評価される

実装内容：
- 「このどちらが好き？」という選択肢
- 「当てはまる？」という質問投げかけ
- 「試してみてほしい」というCTA（行動喚起）
- クイズ形式、引っ張られる演出

**具体例:**
```
インタラクティブ要素:
✅ 「朝食派？夜食派？」→ スワイプして選ぶ
✅ 「このメイク再現してみてください」→ コメントで報告
✅ 「このファッション誰のスタイル似てる？」→ 考えさせる
```

## ⚠️ 注意事項

### Discordの見方
- 🟢 **高リーチ期待**: 保存・シェア要素が多い → すぐ投稿OK
- 🟡 **標準**: そこそこのパフォーマンス期待 → 確認して投稿
- 🔴 **低リーチ**: やや弱い → 修正を検討

### ハッシュタグの最適化
- ❌ 古い実装: 30個のハッシュタグを埋める
- ✅ 新実装: キャプション最初に3～5個の重要キーワード

### キャプション作成のコツ
- ✅ 最初の1～2行が勝負（ここで保存判定される）
- ✅ 「このコンテンツで何が学べるか」を明示
- ✅ 視聴者への質問or CTA を含める
