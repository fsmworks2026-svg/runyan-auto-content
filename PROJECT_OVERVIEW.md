## 📋 プロジェクト背景

### 目標
副業収益化として、**2次元キャラ × Instagram サブスク** で安定収入を作る。

### 背景
伏見さんは経理実務とシステム開発を専門としており、Claude Code と実務スキルを活かしたAIインフルエンサー戦略を検討。

**キーポイント**:
- 顔出し・声出し・動画撮影なし（制約条件）
- 完全自動化で即効性を求める
- グレーな手法は避ける（規約違反・法的リスク回避）

## 📊 ワークフロー

### 自動化パイプライン（毎日午前10時）

```
1️⃣  シナリオ自動生成
    ├─ Claude API で21歳女子大生の日常シーン作成
    ├─ 感情・ムード・舞台設定を決定
    └─ Instagramキャプション案も自動生成

2️⃣  画像生成
    ├─ ChatGPT Image 2.0 で実写感のある「るーにゃ」画像生成
    └─ シーン・服装・表情がシナリオと一致

3️⃣  動画化
    ├─ ElevenLabs Seedance で静止画をアニメーション化
    ├─ BGM・効果音を自動配置
    └─ 字幕（日本語）を焼き込み

4️⃣  Google Drive保存
    ├─ MP4ファイルを保存
    └─ 公開リンク生成

5️⃣  Discord通知
    ├─ プレビュー埋め込み + 動画リンク送信
    ├─ シナリオ内容表示
    └─ キャプション案表示

6️⃣  投稿前確認
    ├─ 伏見さんが Discord で ✅ または ❌ をリアクション
    ├─ ✅ → Instagram 自動投稿予定（Phase 2）
    └─ ❌ → 翌日再生成
```

## 📱 フェーズ分け

### Phase 1（現在）：基盤構築 + 投稿前確認
- シナリオ～Discord通知までの自動化完成
- 投稿前に人間が確認（品質保証）
- **期間**: 5月～6月
- **目標**: 毎日5本のコンテンツ試作

### Phase 2：フォロワー成長戦略
- Instagram 自動投稿機能追加
- 週5～6回の投稿体制
- **目標**: 3ヶ月で1万フォロワー達成

### Phase 3：サブスク開始
- Instagram サブスク機能の有効化（1万フォロワー必須）
- 限定コンテンツ提供開始
- **目標**: 月200～300人のサブスク登録
