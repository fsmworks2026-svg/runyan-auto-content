## リール作成の依頼方法

ユーザーがリール作成を依頼したら、以下の形式で Discord コマンドを出力する。

### 出力フォーマット
```
新規: [シーンの説明]
```

### 例
```
新規: 渋谷のカフェで友達と話しているシーン、ナチュラルメイク
新規: 大学の帰り道、一人でイヤホンしながら歩く、すっぴん
新規: バイト終わりにコンビニ寄る、ちょっと疲れ気味
新規: 休日に部屋でゆっくりしてる、ガチメイクで気合い入れてみた日
```

---

## Discord への貼り付け手順

1. 上の `新規: ...` のテキストをコピー
2. Discord の **#scenario-check** チャンネルに貼り付けて送信
3. 約10分以内にシナリオが自動生成されて Discord に届く
4. 確認して `OK` と返信 → 画像4枚が自動生成される
