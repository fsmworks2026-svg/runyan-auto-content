
### 投稿時刻を変更したい場合
```python
# .github/workflows/runyan-daily.yml 内
on:
  schedule:
    - cron: '0 1 * * *'  # UTC 1:00 = 日本時間 10:00
    
# 日本時間での設定:
# 朝07:00: cron: '0 22 * * *' (前日の22:00 UTC)
# 朝09:00: cron: '0 0 * * *'  (0:00 UTC)
# 夜20:00: cron: '0 11 * * *' (11:00 UTC)
```

## 💡 トラブルシューティング

### Q: Discord 通知に新フィールドが表示されない
**A:** Claude が応答を完全に返していない可能性
```bash
# ログを確認
python main.py 2>&1 | grep -A5 "JSON"

# 再試行
python main.py
```

### Q: 「トレンド対応」になっていない
**A:** ファイルが正しく置き換わっていない可能性
```bash
# 確認
head -10 main.py | grep "トレンド"

# 置き換え再実行
cp main_improved.py main.py
```

### Q: テスト実行が遅い
**A:** 画像生成・動画生成に時間がかかります
```
正常な処理時間:
- シナリオ生成: 10-30秒
- 画像生成: 30-60秒
- 動画生成: 5-10分（ポーリング）
- Google Drive保存: 1-2分
合計: 10-15分
```
